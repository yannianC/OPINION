# 账号异常查询系统

用于查询和分析账号异常情况的Vue 3应用。

## 功能特性

- 时间范围查询：支持选择开始和结束时间进行查询
- 异常账号统计：自动过滤成功任务，只显示失败任务
- 按浏览器ID分组：将同一浏览器的失败任务聚合显示
- 详细信息展示：显示任务主题、失败原因、交易信息等

## 安装和运行

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

应用将在 http://localhost:3001 启动

### 构建生产版本
```bash
npm run build
```

## 使用说明

1. 选择开始时间和结束时间
2. 点击"查询"按钮
3. 查看按浏览器ID分组的异常任务列表
4. 点击"查看"链接可跳转到交易页面
