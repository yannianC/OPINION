# 任务提交系统

这是一个基于Vue 3的任务提交系统，用于向服务器提交任务。

## 功能特性

1. **自动加载映射**：页面加载时自动请求两个API并完成映射
   - `https://sg.bicoin.com.cn/99l/boost/findAccountConfigCache` - 账户配置映射
   - `https://sg.bicoin.com.cn/99l/mission/exchangeConfig` - 交易所配置

2. **手动加载映射**：点击"加载映射"按钮可以主动请求，如果某个请求没有数据，会自动重新请求对应的API

3. **灵活的输入方式**：
   - 浏览器ID输入框支持：
     - 单个编号：`201`
     - 逗号分隔：`201,202,203`
     - 区间：`201-203`
   - 电脑组输入框支持相同的输入方式

4. **单选模式**：浏览器ID和电脑组是单选关系
   - 选择浏览器ID时，提交任务会自动映射对应的电脑组
   - 选择电脑组时，提交任务会自动映射对应的浏览器ID

5. **事件选择**：
   - 默认选择"全部"
   - 支持输入文字筛选事件
   - 支持从下拉列表选择事件

6. **批量提交**：如果有多个浏览器ID，会为每个浏览器ID分别提交一个请求

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

服务器将在 `http://localhost:3001` 启动

### 构建生产版本

```bash
npm run build
```

## 使用说明

1. **加载映射**：
   - 页面加载时会自动加载映射
   - 也可以点击"加载映射"按钮手动加载
   - 映射状态会显示在页面上方

2. **选择输入类型**：
   - 选择"浏览器ID"或"电脑组"单选按钮

3. **输入编号**：
   - 在对应的输入框中输入编号
   - 支持单个、逗号分隔、区间等多种格式
   - 输入后会显示解析结果

4. **选择事件**：
   - 在事件输入框中输入文字进行筛选
   - 或从下拉列表中选择
   - 默认选择"全部"

5. **提交任务**：
   - 点击"提交任务"按钮
   - 系统会为每个浏览器ID分别提交一个请求
   - 提交结果会显示在页面上方

## API说明

### 提交任务API

- **URL**: `https://sg.bicoin.com.cn/99l/mission/add`
- **Method**: POST
- **参数**:
  - `type`: 固定为 `4`
  - `groupNo`: 电脑组编号（通过映射获取）
  - `numberList`: 浏览器ID（单个）
  - `exchangeName`: 固定为 `"OP"`
  - `tp1`: Trending值，如果是全部则为 `"all"`
